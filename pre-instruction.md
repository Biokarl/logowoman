# Инструкция по разработке сайта Logowoman

## Общая информация
- **Проект**: Центр психо-речевой коррекции Logowoman
- **Технологии**: Next.js (React), TypeScript
- **Бэкенд**: Node.js Nest.js (если понадобится)
- **Библиотеки**: Оптимизированные для скролла и анимаций

## Дизайн-система

### Цвета
- **Фон**: Светло-серый (#f5f5f5), и добавить размытые пятна разных размеров акцентных цветов(голубой, розовый), только чтобы не перекрывал основной контент
- **Акцент основной**: #b6eeff (голубой из логотипа)
- **Акцент вторичный**: #ffd1f5 (розовый из логотипа)
- **Активный текст**:rgb(50, 146, 141)
- **Запрещено**: Смайлики Windows, градиенты

### Типографика
- **Шрифт**: system-ui
- **Desktop ширина**: 1440px
- **Скругление**: 12px (кнопки, карточки)

### Адаптивность
- Полностью адаптивный дизайн
- Бургер-меню для мобильных устройств

## Структура страниц

### Главная страница (/)

#### Header
- Логотип в папке public/images/hero/logo с переходом на главную
- Навигация: Услуги(при наведении, чтобы появлялась выпадашка с услугами и небольшим описанием), Специалисты, Сертификаты, Контакты
- Бургер-меню для мобильных (3 полоски, открывается/закрывается по клику)

#### Секция 1: Главный специалист
- Фото главного специалиста public/images/hero/main-specialist.jpg сделать современный красивый вид фото
- Текст о специалисте

#### Секция 2: О центре
- Карточки о центре

#### Секция 3: Команда
- Несколько общих фото специалистов
- Текст (заглушки)

#### Секция 4: Специалисты (карусель)
- Карточки специалистов:
  - Фото
  - ФИО
  - Специальность
  - Опыт работы
  - Иконка сообщения
  - Отзывы (заглушка, позже из Яндекс Карт)
- Размеры: высота 214px, ширина ~510px (адаптивная)
- Функционал: свайп, кнопки навигации (стрелки) появляются только если есть куда листать

#### Секция 5: Видео
- Заглушки для видео

#### Footer
- На усмотрение разработчика

### Страница Услуги (/services)
- Карточки услуг:
  - Фото специалиста
  - Название услуги
  - Описание
  - Специалист, который выполняет

### Страница Специалисты (/specialists)
- Карточки специалистов (крупнее чем на главной)
- При клике открывается модальное окно/детальная страница:
  - Фото
  - ФИО
  - Специальность
  - Опыт работы
  - Иконка сообщения
  - Отзывы (из Яндекс Карт)
  - "Чем может помочь специалист" (список)
  - Образование
  - Сертификаты
  - Отзывы о специалисте

### Страница Сертификаты (/certificates)
- Галерея фото сертификатов (заглушки)

### Страница Контакты (/contacts)
- Контактная информация
- Карта с отметкой центра:
  - Город: Краснодар
  - Улица: Карякина
  - Дом: 27

## Технические требования

### Библиотеки
- Для карусели/свайпа: swiper или react-swiper (оптимизированная версия)
- Для анимаций: framer-motion или react-spring (оптимизированные)
- Для карты: Яндекс Карты API или Leaflet

### Оптимизация
- Использовать Next.js Image для изображений
- Lazy loading для секций
- Оптимизированные библиотеки (tree-shaking)
