# Logowoman - Центр психо-речевой коррекции

Современный веб-сайт для центра психо-речевой коррекции в Краснодаре.

## Технологии

- **Next.js 14** - React фреймворк с App Router
- **TypeScript** - типизация
- **Framer Motion** - анимации
- **Swiper** - карусели
- **CSS Modules** - стилизация

## Структура проекта

```
src/
├── app/                    # Страницы приложения
│   ├── page.tsx           # Главная страница
│   ├── services/          # Страница услуг
│   ├── specialists/       # Страница специалистов
│   ├── certificates/      # Страница сертификатов
│   └── contacts/          # Страница контактов
├── components/
│   ├── Header/            # Шапка сайта
│   ├── Footer/            # Подвал сайта
│   └── sections/          # Секции главной страницы
│       ├── Hero/          # Главный экран
│       ├── About/         # О центре
│       ├── Team/          # Команда
│       ├── Specialists/   # Карусель специалистов
│       ├── Videos/        # Видео
│       └── Appointment/   # Форма записи
```

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен сборки
npm start
```

## Настройка Telegram уведомлений

1. Создайте бота через [@BotFather](https://t.me/BotFather)
2. Получите токен бота
3. Узнайте Chat ID вашего чата/группы
4. Создайте файл `.env.local`:

```env
NEXT_PUBLIC_TELEGRAM_BOT_TOKEN=your_bot_token
NEXT_PUBLIC_TELEGRAM_CHAT_ID=your_chat_id
```
